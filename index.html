json-c API documentation

For general information, see the json-c wiki. There you can also find a link to the download page for source tarballs. For information about building json-c, see the README file.

Versions:

Note: for the very latest development docs clone the source and run "doxygen", then look in the doc directory.
  • Current release: json-c-0.11
    Best starting points: json_object.h or json_tokener.h
  • json-c-0.10 release
    Best starting points: json_object.h or json_tokener.h
  • 0.9 and earlier
You can also look at the tests in the tests directory for some examples.
One of the simplest examples is test2.c.
